# Cadence

A native macOS Pomodoro timer. Stays on your desktop, out of the way.

- 25 minute focus sessions
- 5 minute short breaks (15 min after 4 sessions)
- Space to start/pause
- Menu bar icon to show/hide

No settings. No accounts. Just a timer.

## Install

```bash
./scripts/bundle.sh
sudo cp -R Cadence.app /Applications/
```

Requires macOS 14+.
